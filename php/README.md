BT-Search
=========
简洁舒服的BT资源搜索程序，屌丝必备神器。

###运行环境
* PHP5.2+
* MYSQL 5.1+
* 开启CURL扩展
* 强烈建议启用MYSQL PDO

###程序特点
低调奢华有内涵 高端大气上档次

###示例网站
URL: https://www.bt-ss.com
