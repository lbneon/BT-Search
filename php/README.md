BT-Search
=========
简洁舒服的BT搜索程序，屌丝必备神器。

###运行环境
* PHP5.3+
* 开启CURL扩展
* 保留30MB以上的硬盘空间
(缓存会占用很多，100PV大概生成20M数据)

###程序特点
低调奢华有内涵 高端大气上档次

###示例网站
URL: https://www.bt-ss.com

###安装说明
将所有文件上传到网站根目录，设置cache目录权限为777，并对config.php进行必要修改。
